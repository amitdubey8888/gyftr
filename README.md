# GYFTR

## Quick Start

  Clone the repository:

```bash
$ git clone https://github.com/amitdubey8888/gyftr
```

  Goto project directory:

```bash
$ cd gyftr
```

  Install dependencies:

```bash
$ npm install
```

  Start the app:

```bash
$ npm start
```

  View the app at: http://localhost:3000
